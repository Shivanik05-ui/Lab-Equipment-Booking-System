# Lab-Equipment-Booking-System

## 📌 Project Description
This project is a full-stack web application designed to manage lab equipment bookings efficiently. It allows users to view available equipment, book time slots, and track usage in real-time.

---

## 🚀 Features
- User Login System
- Equipment Listing
- Slot-Based Booking
- Real-Time Availability Check
- Booking History
- Dashboard with Analytics
- Line Chart (Booking Trends)
- Pie Chart (Equipment Usage)

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot
- Database: H2 / MySQL
- Charts: Chart.js

---

## ⚙️ How It Works
1. User logs in
2. Views available equipment
3. Selects date & time slot
4. Booking is saved in database
5. Dashboard updates with analytics

---

## 📊 Dashboard Insights
- Total Bookings
- Today's Bookings
- Booking Trends (Line Chart)
- Equipment Usage (Pie Chart)
